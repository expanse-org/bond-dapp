Connection to an Expanse Node Required

In order to interact with the Expanse network, this dApp must be connected to an Expanse node. Please enter the connection string for the node you wish to connect to below.

It is suggested practice to connect to a local node running on the same system as the dApp. Alternatively, you may connect to a remote node on an internal or external network. Please ensure you research and understand the security best practices related to configuring and running a node or localized wallet.

Suggestion software for nodes are gexp, parity, or exp-lightwallet.

As an additional note, while not required to view your accounts and bonds, in order to deposit, withdraw the account must be unlocked. More information can be found
