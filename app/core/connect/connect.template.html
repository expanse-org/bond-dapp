In order to interact with the Expanse network, this dApp must be connected to a secure Expanse node that is in sync with the current Expanse blockchain. You may connect to an external node, or run a node locally. If you choose to run a local node and have not run an Expanse node before, or your node has not been run recently, it will need to synchronize to the blockchain, this process may take up to an hour and you will be informed as it progresses.
