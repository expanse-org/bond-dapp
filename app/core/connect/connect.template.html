Connection to an Expanse Node Required

In order to interact with the Expanse network, this dApp must be connected to an Expanse node over it's RPC interface. For new users it is suggested practice to connect to a local node running on the same system as the dApp. Alternatively, you may connect to a remote node on an internal or external network. The suggested software to run an expanse node is gexp or parity. Please be sure you research and understand the security best practices related to configuring and running a node or localized wallet in order to ensure the security of your wallet. It is suggested you only provide access to the RPC interface over localhost so only localized dApps can interface with it, although please employ caution as locally viewed websites and dApps may still potentially access it.

As an additional note, while not required to view your accounts and bonds, in order to deposit, withdraw, and transfer the account acting must be unlocked at the time of action. More information can be found in the Managing Accounts section of the users guide.

Example gexp configuration with local RPC interface:
gexp --rpc --rpcaddr localhost
